# 北京林业大学本科生毕业论文\Latex\模板
根据《北京林业大学论文撰写规范及模板》，本科生毕业论文必须符合相关规定。一方面，这些规定不尽不实，存在着相当多的错漏；另一方面，即将毕业的学生实在很难静下来调整论文格式。因此，我们构建了本模板。
